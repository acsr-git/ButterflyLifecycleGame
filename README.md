# ButterflyLifecycleGame
ButterflyLifecycleGame

On behalf of Aiden:

This is the public repo of the Butterfly Lifecycle Game and project created using Scratch (https://scratch.mit.edu/)
by Aiden C for his Oliphant Science Awards 2026.

(1)
index.html contains the packaged game, that was converted from the raw Scratch .sb3 file using TurboWarp Packager (https://packager.turbowarp.org/).

Download and run the index.html file to play the game, or the game can be played at:
https://acsr-git.github.io/ButterflyLifecycleGame/

The code may still be buggy, simply refresh the web browser to restart if needed. 

(2)
"The life cycle of a butterfly v21.sb3" in this repo contains the raw scratch .sb3 game file.
The game can also be played, or the game code blocks can be examined by downloading this .sb3 file and then uploading the file into the Scratch platform/app.

Scratch can be installed and run natively on a PC/laptop: https://scratch.mit.edu/download
or directly from a web browser: https://scratch.mit.edu/projects/editor/?tutorial=getStarted.

If playing the game from Scratch, if encountering bugs, click the stop red button and start green flag repeatedly to restart if needed.

(3)
"LifeCycleOfaButterfly_report.pdf" in this repo contains the report for the game/project.

(4)
A video overview of the game/project can be viewed at:
https://youtu.be/ANbnjztnWIY
